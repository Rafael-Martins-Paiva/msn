# NovaWS

> Projeto backend em Spring Boot com foco na implementação de WebSockets para comunicação em tempo real.

## comandos para executar o projeto

você vai precisar rodar os seguintes comandos

``` bash
mvn spring-boot:run
```

Verifique se você possui:

- **Java 17** (versão usada pelo Spring Boot 3.5.7)
- **Maven 3.8+**
